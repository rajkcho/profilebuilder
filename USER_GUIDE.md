### üíº Deal Structuring

**What it does:** Stock vs cash vs mixed consideration optimizer

**Options:**
1. **100% Cash**
   - Pros: Certainty, no dilution, clean transaction
   - Cons: Requires financing, higher cost, tax inefficient

2. **50/50 Mixed**
   - Pros: Balanced, shared risk/reward, moderate dilution
   - Cons: Complex structuring, some dilution

3. **100% Stock**
   - Pros: Preserve cash, tax-deferred, aligns interests
   - Cons: Dilutive, market risk, potential resistance

**Features:**
- Side-by-side comparison
- Pros/cons analysis
- Visual comparison charts
- Deal value calculations

**Considerations:**
- Buyer's stock price stability
- Seller's tax situation
- Financing availability
- Strategic vs financial buyer

## Tips & Best Practices

### Data Quality

**Good tickers:** AAPL, MSFT, GOOGL, JPM (large-cap, liquid)
**Challenging:** Small-cap, international, OTC (limited data)

**Tip:** If data is missing, try:
- Alternative ticker symbol
- Parent company
- Different exchange listing

### Analysis Workflow

**Recommended order:**
1. Start with **Company Profile** (overview)
2. Run **Comps Analysis** (context)
3. Build **DCF Model** (intrinsic value)
4. If M&A: Use **Merger Analysis** + new M&A tools
5. Export results to PowerPoint

### Performance Optimization

- Data is cached for 5-15 minutes
- Click "Refresh" to force reload
- Close unused tabs for better performance
- Use Chrome/Edge for best experience

## Frequently Asked Questions

**Q: Is the data real-time?**
A: Market data has 15-20 minute delay (free tier). Fundamentals are updated quarterly.

**Q: Can I analyze international stocks?**
A: Yes, but data quality varies. US equities have best coverage.

**Q: How accurate are valuations?**
A: Models are only as good as inputs. Use multiple methods and cross-check.

**Q: Can I save my work?**
A: Export to Excel/PowerPoint. Session state is temporary.

**Q: Is my data private?**
A: Yes. Analysis runs locally in your browser. No data is stored on servers.

**Q: Can I customize the models?**
A: Currently templates are fixed. Future versions may add customization.

**Q: What's the difference between DCF and Comps?**
A: DCF is intrinsic value (fundamental). Comps is relative value (market-based).

**Q: How do I choose between analysis modes?**
A: Start with Company Profile for overview. Use Comps for quick valuation. DCF for detailed modeling. M&A tools for deals.
